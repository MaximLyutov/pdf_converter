Как я автоматизировал обработку инвойсов с помощью Python и сэкономил 12 часов в неделю для отдела ВЭД.

Есть много способов автоматизировать сведение и подготовку данных, один из них — программирование.
Получение от зарубежных поставщиков и обработка инвойсов в другой формат— неизбежная часть работы специалиста в отделе ВЭД.
Обработка прайсов регулярными, корректными и в идеале содержать только ту информацию, которую должен видеть получатель.
Поставщики присылают инвойсы только в pdf-формате (политика безопасности).
И такие данные обрабатывались силами отдела в рукопашную - по несколько монотонных часов работы. Проблема, которую нужно было решить!
    При этом входным требованием было:
  • быстрая и простая обработка инвойсов;
  • простое использование/установка по ОС Windows;
  • возможность доработок;
Для этой цели были взяты несколько утилит из pdf в excel. Предполагалось выгружать все (текстовые) данные в Excel в один, итоговый, файл. Но все эти конвертеры оказались не жизнеспособные под задачи.

На помощь мне пришел Python и его библиотеки для работы с pdf.

Для чтения pdf файлов использовал pdfplumber, а для создания excel файлов openpyxl. Для разбора теста использовал модуль re Для конвертирования скрипта в исполняемый файл – pyinstaller.exe.

Теперь сотрудников на обработку сэкономилось времени порядка 10–12 часов в неделю. Всего получалось, на тот момент, восемь скриптов для обработки брендов:Brembo, Continental, Corteco, Metelli, REINZ, Tenneco и др.

Как пользоваться общая схема (ОС windows)

Вариант 1.
  • Запустить консоль cmd через кнопку Пуск
  • Перейти командой cd в каталог где лежит скрипт и pdf-файл.
  • Ввести название скрипта и название pdf-файла через пробел
Вариант 2.
  • Открыть каталог(и) где лежит pdf-файл и конвертер
  • Перетаскиваете мышью pdf-файл в выполняемый файл (exe) – в результате excel создается там же где pdf-файл.
